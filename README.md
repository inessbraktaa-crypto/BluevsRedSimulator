# Blue vs Red: Cybersecurity Attack-Defense Simulator

## Description
This project is an educational cybersecurity simulator (with a GUI) where a red team attacks and a blue team defends. The goal is to understand cybersecurity attack and defense strategies.


## Features
- Graphical User Interface
- Simulated basic cyberattacks
- Log monitoring and alerts


## Gameplay
- Turn-based combat between Blue Team (player) and Red Team (bot)
- Red Team attacks automatically
- The player chooses a defensive action
- Game Over is automatically detected
- Everything is displayed in logs


## HP System
- Each team starts with 100 HP
- Damage depends on the attack
- Display of current HP on progress bars


## Prerequisites
- Python 3.10 or higher
- Libraries:
    - tkinter
    - Pillow


## How to run
- Open the Blue vs Red Simulator folder
- Open the GUI.py file
- Click Run (or run with Python)
